Plotting with Python

Aug 10, 2018

Drawing a clear and fancy image always means a lot to data visualization as well as information display. Without images, sometimes it could make reading or presenting much more boring and inefficient. So it is no exaggeration to say that mastering this kind of skill is of great importance. Fortunately, Python provides us many powerful tools to plot what we would like to show easily.

In this essay, in order to be more organized, we focus on both different plotting packages as well as different image types. The packages that need installation in advance include matplotlib, seaborn and pyecharts.
