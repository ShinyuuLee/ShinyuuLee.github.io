Linux Basic Operation

Nov 30, 2017

This essay focuses on some basic Linux command statements at the terminal. When we use Linux system, it’s essential for us to know some of them, especially when we can only operate at the terminal. There are many introductory documents or professional websites on the Internet, and I strongly recommend http://www.runoob.com/linux/linux-command-manual.html. If you encounter any puzzled things, try to find the answers by yourself can be a useful experience.

ls

The command statement “ls” is used to display the contents of the specified working directory. In fact, we can achieve a lot of specific functions by adding some words. For example, if we add ‘-t’, the files would be listed according to the order of establishment time. With commend “-l”, in addition to the file name, but also the file type, permissions, owner, file size and other details would be listed. “-a” could help us view all files and directories, including these hidden files whose names begin with ‘.’ .

find

The Linux “find” command is used to find the file in the specified directory. This command can help us quickly find the needed documents which have certain characteristics. For instance, we can use “$ find . -ctime -20” to view all files in the current directory and its subdirectories that have been updated in the last 20 days.

File Management

The basic functions of managing files include “copy”, “move” and “delete”. Correspondingly, at the Linux terminal, we can realize them through commands “cp”, “mv”, “rm” respectively. In addition, if we would like to view the content of a document, we could make it by entering “cat” and the path of the document.

Run Code & Edit

Here I use a python code document “test.py” as an example to show how we run and modify it at the Linux terminal. When we enter “ex test.py”, then there comes to the second interface, where we should enter “visual” to see the content. After succeeding to see the third surface, we can click the i key on the keyboard, then “–INSERT–” appears at the bottom, which means we are allowed to edit the code. After editing, we should press the ESC key and enter “:wq” to save and exit. Actually, this method also applies to edit documents of other types such as “.txt”.




System Management

The linux command “who” is used to show which users are on the system. “top” can help to display the real time condition of system process just like the third picture below, and when we want to close some programs, use “kill” command to manage the system. What’s more, if we want to get more detailed information about any command, use “man” to find them.
