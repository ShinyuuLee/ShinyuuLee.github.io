Variable

In simple terms, variables are the most basic storage units in programming. You can use “=” to put something into a variable, and use print() to view it. The basic data types in Python are integer(int), float, str, bool and NoneType.

Then let’s talk about the mathematical operations. In addition to ordinary addition(+), subtraction(-) and multiplication(*), Python also provides a lot of other methods to help us quickly calculate.

Container

After the initial experience of python, we first introduce the following four containers to hold variables: list, dictionary, tuple and set. Among them, the elements inside list are ordered and changeable, and list can hold any object in Python.

Here we come to introduce the usage of dictionary. The data in the dictionary must appear as a key-value pair and logically, the keys are not allowed to be repeatable. In addition, the keys are often unchangeable.

Tuple can be understood as a fixed version of the list, which elements inside can not be modified. The statement to create a tuple is as same as list, with only [ ] changing into ( ). And set can be created by using { } with .add and .discard to change elements inside.

There is also a very important skill here which I think need to be mentioned. Slice can really help you a lot when you write a code. Taking a part of list or tuple is a very common operation, without loop, slice in Python could greatly simplify this operation.

Judgement

There are only two types of Boolean data: True and False. Human beings judge the truth with authenticity, while the computer only knows 1 and 0. So learn how to use boolean expressions is important for us before we write a loop.

We also should know membership & identify operators. Is and is not are identify operators, while in and not in are membership ones.

Actually, any object in Python can determine its boolean value, you could use bool function to have a try by yourself, such as bool([ ]) to have a look. And also try to put in “not [ ]”, and you will find the return value is ‘True’. Just think about it.

After “if” or “else” you write down the condition (which we introduced in Judgement), and then you write down “:” and something which you wanna python to do for you. We could use “elif” when there are many conditions need to be judged.

Class

Here we briefly introduce the concept of Polymorphism, Encapsulation and Inheritance. First we use the add() function to explain what is polymorphism.

Although it seems very simple, but the key is that the parameters can be any object which supports additions. Actually, len(), type() and isinstance() all have this feature. I believe it is easy to understand the concept of polymorphism now: the provision of a single interface to entities of different types.

Then we try to understand Encapsulation through the realization of “Class”. Here I want to mention Object Oriented Programming(OOP), because Python is one of the OOP languages. OOP is a programming paradigm based on the concept of “objects”, which may contain data, in the form of fields, often known as attributes; and code, in the form of procedures, often known as methods.
